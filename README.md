# 贪吃蛇游戏

## 如何运行
1. 编译所有Java文件：
   ```
   javac SnakeGame.java GamePanel.java Snake.java Food.java
   ```

2. 运行游戏：
   ```
   java SnakeGame
   ```

## 游戏控制
- 使用方向键（↑↓←→）控制蛇的移动方向
- 吃到红色食物可以增长蛇身并增加得分
- 撞到墙壁或自己的身体会导致游戏结束

## 重新开始游戏
- 游戏结束后，按空格键可以重新开始游戏

## 游戏特点
- 简单易玩的贪吃蛇游戏
- 实时显示得分
- 游戏结束后显示最终得分
- 支持无限次重新开始
